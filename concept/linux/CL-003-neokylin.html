# L003 中标麒麟（NeoKylin）

中标麒麟（NeoKylin）
创建时间： 2021.10.23
最后更新： 2021.10.24
竢实扬华 · 自强不息

# 1 麒麟家族

麒麟家族包括中标麒麟（NeoKylin）、银河麒麟（Kylin）、优麒麟（Ubuntu Kylin）和麒麟信安（Kylinsec）。其中中标麒麟和银河麒麟占主要地位。

麒麟系操作系统在信创政府领域内具备较大优势，中标麒麟的市场占有率超过单三成，多年来一直稳居第一。

# 2 中标麒麟

# 2.1 官方介绍

2010年，中标软件与国防科学技术大学联合推出“中标麒麟”操作系统产品。

中标麒麟操作系统系列产品主要以操作系统技术为核心，重点打造自主可控、安全可靠等差异化特性。

该系统是由民用的“中标Linux”操作系统和军用“银河麒麟”操作系统合并而来，最终以“中标麒麟”的新品牌统一出现在市场。

该系统还针对X86及龙芯、申威、众志、飞腾等国产CPU平台进行自主开发，率先实现了对X86及国产CPU平台的支持。

# 2.2 网上的一些评价

  • 中标麒麟为的不是完全自主研发，而是为了将系统组件把控在自己手里，对每个系统组件进行审计，以达成可控可管。这样能保证重要企业操作系统的安全性。

  • 中标麒麟基于RHEL/CentOS，对于很多新软件、新技术的支持其实是很慢甚至根本不考虑的。可能导致使用新技术、新方法的公司，没有市场推广自己的产品，甚至因为自己的技术“太新”而无法用在“稳定”发行版上面。

  • 专业漏洞扫描出的高危漏洞（很多是软件和内核的安全漏洞），无安全更新可用。
