# 3 lag()与lead()

lag()与lead()
创建时间： 2021.04.01
竢实扬华 · 自强不息

lag和lead分析函数可以在同一次查询中取出同一字段的前n行的数据(lag)和后n行的数据(lead)作为独立的列，从而更方便地进行进行数据过滤。

这种操作可以代替表的自联接，并且lag和lead有更高的效率。

语法：

lag(exp_str, offset, def_val) over()

lead(exp_str, offset, def_val) over()

其中：

  • exp_str为要取的列字段

  • offset为取偏移后的第几行数据

  • def_val为没有符合条件的默认值

  • over()表示lag()与lead()操作的数据都在over()的范围内，over()里可以使用partition by（用于分组）和order by（用于排序）

使用样例：

select t.id,
       lag(t.id, 1, null) over(order by t.id asc) pre_id,
       lead(t.id, 1, null) over(order by t.id asc) next_id
  from t;
