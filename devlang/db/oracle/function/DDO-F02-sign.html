# 2 sign()

sign()
创建时间： 2021.03.31
竢实扬华 · 自强不息

# 2.1 基本用法

sign()函数只有一个参数，函数返回参数的正负性。

  • 若返回1，表示参数大于0

  • 若返回-1，表示参数小于0

  • 若返回0，表示参数等于0

# 2.2 用于比较大小

利用sign()函数的特性，可以对两个数值进行大小比较，也可以获取两个数值中的最大最小值，获取最小值写法如下：

select decode(sign(变量1 - 变量2), -1, 变量1, 变量2) from dual;
