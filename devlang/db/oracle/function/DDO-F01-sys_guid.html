# 1 sys_guid()

sys_guid()
创建时间： 2021.03.23
最后更新： 2021.03.24
竢实扬华 · 自强不息

# 1.1 基本用法

用于生成UUID（32位的不重复数据，生成的依据主要是时间与机器码，具有世界唯一性）。

select sys_guid() from dual;
1

该函数的特点（存在的问题）：

  • 返回值类型为RAW

  • 返回值中没有分隔符

  • 返回值中的字母全大写

关于RAW

RAW是Oracle用于保存位串的数据类型，是可变长度的二进制类型（目前已被LOB替代），声明方式RAW(L)，其中L为长度，作为数据库列最大2000字节，作为变量最大32767字节。

RAW类型的好处是在网络中的计算机之间传输RAW数据时，或者使用Oracle应用程序将RAW数据从一个数据库迁移到另一个数据库时，Oracle服务器不需要执行字符集转换。

# 1.2 符合RFC4122标准的UUID

RFC4122中定义的UUID标准格式型如“8-4-4-4-12”，如：a8f662b8-6e7b-13fe-e040-970a437c6bd7。

CREATE OR REPLACE FUNCTION GET_UUID RETURN VARCHAR IS
  V_GUID VARCHAR(50);
BEGIN
  V_GUID := LOWER(RAWTOHEX(SYS_GUID()));
  RETURN SUBSTR(V_GUID, 1, 8) || '-' || SUBSTR(V_GUID, 9, 4) || '-' || SUBSTR(V_GUID, 13, 4) || '-' 
         || SUBSTR(V_GUID, 17, 4) || '-' || SUBSTR(V_GUID, 21, 12);
END GET_UUID;
