# 1 数据去重

数据去重
创建时间： 2021.03.31
竢实扬华 · 自强不息

数据去重，首先要明确能够定义重复的字段范围，样例表中的主键id就不作为判断数据是否重复的条件：

（1）查看目标表当前记录总数：

select count(1) from 目标表 t;
1

（2）建立备份表：

create table 目标表_bak as (select * from 目标表);
1

安全第一！

进行数据处理前，一定要对目标表进行备份，以确保数据可回退。

（3）根据判定数据是否重复的字段，建立临时表（重点是要根据判定数据是否重复的字段进行分组，且记录每组数据的最大rowid）：

create table 目标表_tmp as 
(select t.字段1, t.字段2, t.字段3, ..., t.字段n, max(t.rowid) as dataid from 目标表 t 
group by t.字段1, t.字段2, t.字段3, ..., t.字段n);

（4）将rowid不在临时表中的目标表的数据删除：

delete from 目标表 t where not exists (select 1 from 目标表_tmp where dataid = t.rowid);
1

（5）查看目标表删除重复数据后的记录总数：

select count(1) from 目标表 t;
