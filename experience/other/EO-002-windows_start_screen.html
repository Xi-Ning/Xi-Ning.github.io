# 002 Win10将文件的快捷方式固定到开始屏幕

Win10将文件的快捷方式固定到开始屏幕
创建时间： 2021.12.20
竢实扬华 · 自强不息

想要将一些常用的文件添加到开始屏幕中，效果如下图：

设置方法如下：

  1. 在文件上右键，【发送到】 -> 【桌面快捷方式】。

  2. 在桌面上找到刚刚生成的快捷方式，将其名称修改为想要的名称。

  3. 将快捷方式拷贝到C:\ProgramData\Microsoft\Windows\Start Menu\Programs目录下，完成后就可以在开始菜单中找到该快捷方式了：

  1. 在开始菜单中快捷方式上，右键，【固定到“开始”屏幕】。
