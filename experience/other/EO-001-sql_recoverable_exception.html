# 001 SQLRecoverableException

SQLRecoverableException
创建时间： 2021.04.06
竢实扬华 · 自强不息

新换笔记本，搭建好开发环境后，应用启动失败，无法连接数据库，详细报错信息如下：

原因是windows账户名为中文而导致，解决办法：通过设置VM arguments，将java应用所使用的系统属性“用户名”修改为英文名，如：-Duser.name=xxx。

在sts中的设置步骤如下：

Run -> Run Configurations...（或Debug Configurations...） -> 选中要启动的应用 -> 切换Arguments标签页 -> 在VM arguments中添加 -Duser.name=xxx
