# 1 单词

  • collection：集合
  • synchronized [ˈsɪŋkrənaɪzd]：同步的

  • reduce：减少；缩小；使变成
  • tail：跟踪；尾部
  • comparator：比较器
  • filter：过滤器；筛选
  • predicate：谓词；断言
  • transaction：交易；处理；业务
  • parallel [ˈpærəlel]：并行的
  • optional：可选择的
  • expression：表达
  • statement：陈述；声明
  • simplify：简化
  • handler：操作者；处理者
  • callable：可调用的
  • privileged：有特权的
  • process：过程
  • execute：执行；实施
  • multiple：数量多的；多种多样的；倍数
  • accept：接受；同意
  • closure：闭包；使关闭
  • dump：转存；扔掉
  • stack：堆栈；大量；一堆
  • reverse：使反转；相反的
  • negate：否定；取消；使无效
  • compose：组成
  • pipe：管道
  • integrate：整合；使成为一体
  • boxing / unboxing：装箱 / 拆箱
  • fork / join：分支 / 合并
  • consumer / supplier：消费者 / 供应商
  • unary / binary：一元的 / 二元的

  • currency：货币
  • tricky：棘手的

  • execute around：环绕执行

# 3 缩写

  • JDK (Java Development Kit)：Java 发展工具
  • JVM (Java Virtual Machine)：Java 虚拟机
  • CPU (Central Processing Unit)：中央处理器
  • SQL (Structured Query Language)：结构化查询语言
  • API (Application Program Interface)：应用程序接口
  • GUI (Graphical User Interface)：图形用户界面
  • DRY (Don't Repeat Yourself)：不要重复自己（一种软件工程原则）
